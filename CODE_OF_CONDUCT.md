# Code of Conduct

## Our Standards
This project aims to maintain a simple, respectful, and collaborative environment.

We expect contributors to:
- Be respectful in communication
- Avoid offensive or toxic language
- Focus on constructive feedback
- Respect different levels of experience
- Keep discussions relevant to the project

## Unacceptable Behavior
- Harassment or discrimination
- Insulting or aggressive language
- Spamming or irrelevant issues
- Disruptive behavior in discussions or pull requests

## Enforcement
Project maintainers may remove, block, or restrict any contribution or interaction that violates this Code of Conduct.

## Scope
This Code of Conduct applies to all project spaces including issues, pull requests, and discussions.
