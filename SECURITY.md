# Security Policy

## Reporting a Vulnerability

If you find a security issue in this project, please report it responsibly.

You can open a GitHub issue or contact the maintainer directly at "mr.az.original@gmail.com"

Please include:
- Description of the problem
- Steps to reproduce it
- Any relevant details or screenshots

## Response Time

We will try to respond as soon as possible and fix valid issues in a reasonable timeframe.

## Notes

This project is mainly for educational and personal use, so security may not be as strict as production software. However, responsible disclosure is always appreciated.
